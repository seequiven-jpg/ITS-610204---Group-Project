**Summary Results of SPM-Warisan Project**


**What we achieved**

Our team is really happy and proud that we finished our screens for the Junior, Senior and Admin actors.Every features we talked about in our User Stories is now in our Figma Design.Even though they are just screens and ideas,it shows the full journey of how a junior can find a book, chat with a senior and so on.

**Our Testing Results**

Because the app isn't real working one, we did a self-check by acting as the user ourselves.We followed our manual test cases to make sure all the features and the screens UI make sense.

**How we feel about the project**

As a leader, I have to be very honest that this project was a huge challenge for us. We actually started this project aftet the midterm, so we had less than two weeks to rush everything.At the start,I felt so stressed because I couldn't really understand what the project was about, everything was new for us like using the Github and Trello for the first time. It was a lot to learn in such a short time.

But even with the stress, we worked hard to catch up.It was a really great feeling to see how our ideas for this project into a real-looking app.We learned a lot about working together as well as getting feedback from our teacher, and we also proud that we finally made it here.Hopefully one day in the future, we could actually build something that help Malaysian students.
